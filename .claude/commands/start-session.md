Bắt đầu session làm việc cho ICare247. Thực hiện đúng thứ tự:

1. Đọc `.claude/memory/last_session.md` để biết session trước làm gì
2. Đọc `.claude/memory/project_current_phase.md` để biết phase hiện tại
3. Đọc `TASKS.md` để xem task nào đang In Progress / Todo
4. Tóm tắt ngắn gọn cho user:
   - Session trước làm gì
   - Task đang dở (nếu có)
   - 3 task tiếp theo nên làm (theo thứ tự ưu tiên)
5. Hỏi user: "Hôm nay làm task nào?"
