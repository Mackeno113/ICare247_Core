# Rule — i18n bắt buộc (mọi text hiển thị)

> **Ràng buộc bất biến:** KHÔNG chuỗi hiển thị nào được hardcode. Mọi nhãn/tiêu đề/nút/thông báo/placeholder/
> tooltip/cột phải đi qua i18n. Vi phạm = lỗi review, không merge.

## Hai cơ chế — chọn đúng nguồn

| Loại chuỗi | Cơ chế | Ghi chú |
|---|---|---|
| **Màn/control viết tay** (shell, nút, dialog, header bespoke) | **`Loc.L(key, "fallback vi")`** (LocalizationService, JSON overlay client) | Phần lớn UI admin nằm ở đây |
| **Nội dung động cấu hình từ WPF** (label/placeholder/tooltip field, caption cột View, title form) | Backend resolve `Sys_Resource` theo `Lang_Code` (MetadataEngine) — **đã có sẵn** | KHÔNG bọc `Loc.L` lại |

## Cách dùng `Loc.L` (chuỗi hand-coded)

1. Component **kế thừa** `@inherits LocalizedComponentBase` (có sẵn `Loc`).
2. Mọi chuỗi: `@Loc.L("scope.category.name", "Văn bản tiếng Việt mặc định")`.
   - Tham số: `Loc.L("view.footer.count", "Hiển thị {0} / {1}", n, total)`.
3. **KEY suy từ cấu trúc màn, JSON chỉ là value overlay** — KHÔNG gõ key tay vào JSON.
   Thiếu bản dịch → fallback chuỗi vi tại chỗ gọi → "key có trước, dịch sau".

## Quy ước key (spec 10)

- Dùng chung: `common.action.*` (save/cancel/create/edit/delete/refresh…), `common.filter.*`,
  `common.validation.*`, `common.search.placeholder`, `common.column.actions`, `common.label.error`.
- Namespace riêng màn: `{scope}.{category}.{name}` — vd `masterdata.list.manage`, `view.footer.count`,
  `organization.company.section.identity`.
- Nav: `nav.module.{key}`, `nav.screen.{module}.{screen}`, `nav.group.{key}` (suy từ slug).

## Cấm & lưu ý

- ❌ Chuỗi tiếng Việt/Anh cứng trong markup hoặc `@code` trả ra UI.
- ❌ Tự thêm key vào `en.json` thủ công (en.json = overlay, để rỗng tới khi dịch).
- Chuỗi fallback tĩnh trong code C# (vd `AppNav` title) → đánh dấu `i18n:skip-hardcode` cho scanner,
  và **phải** render qua `Loc.L(key, title)` ở nơi hiển thị.
- Số/ngày/tiền: set `CultureInfo` (LocalizationService đã lo) → DevExpress format tự đổi theo locale.
- Kiểm nhanh: bật **pseudo-loc** (`qps`) → chuỗi nào KHÔNG bọc ngoặc `⟦…⟧` là còn hardcode.
